Building a Class Diagram

Step #4: Creating a Class and Name it

To add a new class to the diagram, click on the class icon in the toolbar, and then click anywhere in the Editor pane (top-right). See the figure.
To name this class "Course", you go to the Properties tab in the Details pane (bottom-right) and enter the name in the name field. Notice how the meta-model in the Explorer (top-left) is updated automatically.
