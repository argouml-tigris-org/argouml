Known Incompatibilities

Here is a list of features which are not implemented yet and can be seen as a violation of the UML specification (1.4).

All Diagrams

  • Graphical representation of Qualifiers is missing.
  • Optional directional triangle missing on association names.

Class Diagram

  • Parameterized classes are missing.
  • Cannot show collaborations on a class diagram.

Object Diagrams

  • Currently not implemented as a separate diagram, use a Deploymentdiagram.
  • Attributes cannot have attribute values.

Activity Diagrams

  • Swimlanes missing.
  • SubActivity states are missing.
  • Signal receipt/sending missing.
