Developer Zone (15 Oktober 2003)

ArgoUML is not only a free UML modelling tool, it is also an Open Source Development project where you are invited to contribute.

This part of the ArgoUML homepage is dedicated to the project members that are contributing or the persons that would like to become project members and contribute. If you are interested in how the ArgoUML project works and how we do things you are also welcome.

Goals

The goal of the ArgoUML project is to build an object oriented design tool that is:
  • a joy to use
  • actually helpful to designers when they are making design decisions, by offering cognitive support through critics
  • completely open source Java (FreeBSD license)
  • supporting UML version 1.3
  • modular and extensible
  • integrated with the web and other Tigris tools.
You can also read the Project Vision and read the list of features to learn more about what ArgoUML is capable of already.

As of October 2002 developers of ArgoUML can use RefactorIT, a java based tool to assist in refactoring and code analysis. Just register, download and request the opensource license, which allows you to work with a number of Open Source projects and up to 50 of your own classes, which would even allow you to work on smaller modules outside the ArgoUML namespace.

Thanks to RefactorIT for their support.
