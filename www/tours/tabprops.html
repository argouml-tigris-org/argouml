ArgoUML Properties Tab

ArgoUML's Properties Tab shows the properties of the selected design element. The contents of the tab pane vary depending on the type of design element selected. Property panes are defined for all different design elements. They also contain certain toolbuttons: e.g. in the example above the properties tab for a Class is shown, which has (amongst others) toolbuttons for the creation of attributes and operations.
