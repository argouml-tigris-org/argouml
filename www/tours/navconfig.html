ArgoUML Configure Perspectives Window

 

Argo's Configure Perspectives Window allows you to change the content and appearence the Navigation Pane. This window is made up of two tabs: Perspectives and Panes.

  • Perspectives Tab:
    This tab allows you to change the perspectives that deterime the contents of the Navigation Pane. To see the rules that make up a perspective, select the perspective from the list at the top of the window. To add a rule to the selected perspective, select the rule from the library (lower left) and click on the ">>" button. To remove a rule from the perspective, select it in the selected rules list (lower right) and click on the "<<" button.
  • Panes Tab:
    This tab allows you to divide the Navigation Pane into two or three parts and specify the relationship between these parts. Note: this is not implemented yet.
