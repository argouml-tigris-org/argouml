Old outdated page about how to access Project ArgoUML Source Code using cvs. The information you need is in the servlet project source page.

ArgoUML CVS Access:

CVS Repository:

To access the CVS source code repository use the following CVSROOT:
   :pserver:guest@cvs.tigris.org:/cvs

You may also use the login "guest" with password "guest" to gain read-only CVS access. If you have been given write access to the CVS repository, then replace "guest@cvs.tigris.org" with your own login (e.g., "my_login@cvs.tigris.org") and use your own tigris user password.

The modules to checkout are:

  • argouml: The whole project, including the project web pages
  • argouml/src: Java source code ("stable" tree) v 0.7, in branch "argo-nsuml" you find the source for version 0.8
  • argouml/src_new: Java source code (unstable development tree)
  • argouml/www: the project web site (argouml.tigris.org).

The way most people contribute changes is to email a "patch" to one of the "module owners". These are developers who are responsible for certain sections of the code (called modules). There is a list of module owners here. Once a module owner sees your contribibution, he or she will review it and integrate it into the source code repository.

For more information on using CVS, documentation, and CVS clients, click on the "Tool Docs" menu at left and select the "Source/CVS" option. Or, click here.
