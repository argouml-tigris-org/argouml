Counting downloads of ArgoUML

Downloads of ArgoUML 0.18.1

ArgoUML 0.22 was released on 8th of August 2006.

The count is only from the 18th of September. Before that the counting was not enabled.

When zip tar.gz appb Total DLs jws ls Total jws Manpdf Manidx
September 2006 3983 346 100 4429 2213 56 2269 492 1760 (17 days missing)
October 2006 10792 903 245 11940 5651 164 5815 1180 3602

The headers above mean:

  • zip - zip downloads.
  • tar.gz - tar downloads from argouml-downloads page.
  • appb - appbund downloads from argouml-downloads page.
  • jws - java web start starts.
  • ls - java web start starts of latest stable.
  • Manpdf - Number of pdf downloads of the manual.
  • Manidx - Number of views of the first page of the user manual (from the argouml-stats project).

How do we count?

The counts are based on Pageviews from Google Analytics. We count the click of links to .zip or .tar.gz files of the release using an onClick javascript command. If the browser doesn't do this, the download or start is not counted. The way Google Analytics work we also count failed attempts to do downloads. There is not much we can do about it. For Java Web Start we count the amount of Pageviews of the jnlp file.
