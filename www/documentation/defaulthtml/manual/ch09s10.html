9.10. The Tools Menu

This menu provides a generic menu attachment point for any plug-ins provided with ArgoUML. The standard system has one plug-in, and this menu entry contains one submenu by default.

9.10.1. Export as XMI…

This menu-item allows to save the complete structure of the UML model as a XMI file, according the XMI V1.0 standard. Beware the fact that this file will only contain the model, not any diagram layout. Hence, if the xmi file is reloaded with the File - Open Project… menu, then the diagrams are lost.

When the menu is activated, the standard filechooser appears, see Figure 9.20.

Figure 9.20. The dialog for Export XMI….

Warning

The “.xmi” extension is not added automatically. You have to enter it with the filename.
