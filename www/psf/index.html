Team Project Set Files for ArgoUML

Here are the files used to download the correct set of files from the Subversion repository when developing ArgoUML with Eclipse.

argouml-core-projectset.psf Main file for developing ArgoUML.
argouml-modules-projectset.psf File with all modules. Requires argouml-core-projectset.psf.
argoeclipse_projectset.psf For developing ArgoEclipse. Requires argouml-core-projectset.psf.
argoprint_projectset.psf For developing ArgoPrint. Requires argouml-core-projectset.psf.
argouml-sequence.psf For developing sequence diagrams. Requires argouml-core-projectset.psf.
gef-swidgets-toolbar-projectSet.psf For downloading the source of some of the used tools.
argouml-doc-projectset.psf For working with the documentation. Requires argouml-core-projectset.psf.
