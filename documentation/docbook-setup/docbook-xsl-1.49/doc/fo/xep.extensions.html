Name

xep.extensions — Enable XEP extensions?

Synopsis

§39: §133

  1| <xsl:param name="xep.extensions"
  2|            select="0"/>

Description

If non-zero, XEP extensions will be used. XEP extensions consists of PDF bookmarks and document information.
