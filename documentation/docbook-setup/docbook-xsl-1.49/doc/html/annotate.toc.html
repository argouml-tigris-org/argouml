Name

annotate.toc — Annotate the Table of Contents?

Synopsis

§17: §148

  1| <xsl:param name="annotate.toc"
  2|            select="1"/>

Description

If true, TOCs will be annotated. At present, this just means that the RefPurpose of RefEntry TOC entries will be displayed.
